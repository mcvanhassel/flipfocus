How to Generate Your OG Image

This template creates a 1200x630 pixel OG image perfect for social media sharing.

Method 1: Browser Screenshot (Easiest)

  1. Open this HTML file in Chrome or Firefox
  2. Press F12 to open Developer Tools
  3. Click the device toolbar icon (mobile/tablet view)
  4. Set dimensions to 1200x630
  5. Right-click on the blue gradient area and select "Capture screenshot"
  6. Save as flipfocus-og-image.png
  7. Move the file to assets/images/ directory

Method 2: Online Screenshot Tools

  1. Upload this HTML file to services like:
    • htmlcsstoimage.com
    • websitetoimage.com
    • web-capture.net
  2. Set dimensions to 1200x630
  3. Generate and download the image

Method 3: Puppeteer/Playwright (Advanced)

If you're comfortable with Node.js, you can use automated screenshot tools.
